## 1. 简易哈希表

### １．lcNo:242 有效的字母异位词

题目：给定两个字符串 `s`和 `t`，编写一个函数来判断 `t`是否是 `s`的字母异位（注：若 `s`和 `t`中每个字符出现的次数都相同，则称 `s`和 `t`互为字母异位词。）

> 　　该问题明显可以用hash完成，但是这个时候一个简单数组就可以达到哈希效果。
>
> 代码如下：
>
> ```cpp
> bool isAnagram(string s, string t) {
>         int record[26]={0};
>         for(char c:s){
>           record[c-'a']++;
>         }
>         for(char c:t){
>           record[c-'a']--;
>         }
>         for(int i=0;i<26;i++){
>           if(record[i]) return false;
>         }
>         return true;
>     }
> ```
